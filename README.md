# E&I Calc — เครื่องคำนวณงานระบบไฟฟ้า (PWA)

เครื่องมือคำนวณงานระบบไฟฟ้า/เครื่องมือวัด ตามมาตรฐาน วสท./IEC
รันเป็น **Progressive Web App** — ติดตั้งลงหน้าจอ iPhone ได้เหมือนแอปจริง เปิดเต็มจอ ใช้งานออฟไลน์ได้

## โมดูลคำนวณ (v1.0)
1. **สาย & CB** — หาขนาด CB (I_N) และ ampacity สายที่ต้องการ พร้อม derating Ca·Cg·Ci
2. **แรงดันตก** — %VD จากค่า mV/A/m พร้อมเช็คเกณฑ์ ≤ 5%
3. **มอเตอร์** — FLC 3 เฟส + กระแสสตาร์ท (DOL / Y-Δ / Soft / VSD)
4. **กราวด์** — ความต้านทานหลักดิน (Dwight) + กลุ่มแท่ง เช็คเกณฑ์ ≤ 5 Ω

## วิธีเอาขึ้น iPhone

### ทางที่ 1 — ลองเร็วสุด (ไม่ต้องโฮสต์)
ส่งไฟล์ `index.html` เข้า iPhone (AirDrop / อีเมล / iCloud Drive) แล้วเปิดใน Safari
> ข้อจำกัด: โหมดนี้ยังไม่ทำงานออฟไลน์เต็มรูปแบบ (service worker ต้องการ https)

### ทางที่ 2 — โฮสต์ฟรี แล้ว "เพิ่มลงหน้าจอโฮม" (แนะนำ ✅)
1. อัปโหลดทั้งโฟลเดอร์ขึ้นโฮสต์ฟรีตัวใดตัวหนึ่ง:
   - **Netlify Drop** — ลากโฟลเดอร์วางที่ https://app.netlify.com/drop ได้ลิงก์ทันที
   - **GitHub Pages** — push ขึ้น repo แล้วเปิด Pages
   - **Cloudflare Pages / Vercel** — ลากโฟลเดอร์อัปโหลด
2. เปิดลิงก์ที่ได้ใน **Safari** บน iPhone
3. กดปุ่ม **แชร์** (สี่เหลี่ยมมีลูกศรขึ้น) → **เพิ่มลงในหน้าจอโฮม (Add to Home Screen)**
4. จะได้ไอคอนแอปบนหน้าจอ เปิดแล้วเต็มจอ ไม่มีแถบ Safari — และใช้งานออฟไลน์ได้

> ต้องเป็น **Safari** เท่านั้น (Chrome บน iOS ยังเพิ่มลงโฮมสกรีนไม่ได้)

## โครงสร้างไฟล์
```
eecalc/
├── index.html      ← ตัวแอป (โค้ด+สูตรทั้งหมดอยู่ในไฟล์เดียว)
├── manifest.json   ← ข้อมูลแอปสำหรับ PWA
├── sw.js           ← service worker (ออฟไลน์)
├── icon-180.png    ← apple touch icon
├── icon-192.png
├── icon-512.png
└── README.md
```

## แก้ไข/ต่อยอด
สูตรทั้งหมดอยู่ใน `<script>` ท้ายไฟล์ `index.html` (ฟังก์ชัน `calcCB`, `calcVD`, `calcMtr`, `calcGnd`)
เพิ่มโมดูลใหม่ = เพิ่มปุ่ม tab + section + ฟังก์ชัน แล้ว bind input
